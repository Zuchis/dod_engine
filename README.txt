Building and Running Instructions:

1. To build the program simply run:
    $ make

2. The executable generated is named "magic", just run it with no arguments:
    $ ./magic


Simulation Commands:

    - "w" and "s" keys move the pieces up or down relatively to the ground

    - "up" and "down" keys move the ground on the Y axis

    - "left" and "right" keys move the ground on the X axis

    - "q" and "e" keys rotate the ground around the -X axis

    - move the mouse with the left button pressed rotate the camera around the origin

    - the "g" key animate the pieces, pressing it again will revert the animation
